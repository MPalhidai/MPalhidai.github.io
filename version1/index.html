Temperature Conversion

This ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 


      def c_to_f(temp)

      return temp * 9/5 + 32

      end

      print "Enter degrees in Celcius: "
      temp = gets.to_f
      puts "The temperature is #{c_to_f(temp)} degrees Fahrenheit."

Foobar

This ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 


      print "How many digits of the Foobar pattern do you want to see: "
      foobar_iteration = gets.to_i

      foobar = []

      (1..foobar_iteration).each do |index|
        foobar << index
      end

      foobar.each do |index|
        foo = index % 3
        bar = index % 5
        if foo == 0 && bar == 0
          puts "Foobar"
        elsif foo == 0
          puts "Foo"
        elsif bar == 0
          puts "Bar"
        else
          puts index
        end
      end
