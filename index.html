My Expertise

I've got the passion to learn them all, but for now...

Sass

I like to make ample use of CSS preprocessors. Sass allows me to make beautiful and dynamic objects that take a bland interface and makes it aesthetically pleasing.

React/Redux

I don't have enough good words to describe modern JS libraries. The virtual DOM makes interfaces lightening fast. I use React and Redux to organize modular components in a neat state management system.

Rails API

For a backend I use a Rails API for its simplicity. If it only needs to store and serve data, a service-oriented architecture is the way to go. When it’s ready I like Heroku and AWS for deployment.
